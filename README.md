# Dayflow - Modern HR Management System

A comprehensive HR management system for streamlined workforce operations.

## Features
- Employee Management
- Payroll Processing
- Leave Management
- Project Tracking
- Client Management
- Asset Management
- Dashboard Analytics

## Installation
1. Copy files to your web server
2. Import the database from Database/dayflow.sql
3. Configure database connection in includes/config.php
4. Access via web browser
